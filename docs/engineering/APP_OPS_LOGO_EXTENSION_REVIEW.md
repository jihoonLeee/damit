# App Ops Logo Extension Review

Date: 2026-03-26
Owner: PM

## Decision

`/app`와 `/ops` 헤더에도 `DAMIT` lockup을 확장 적용했다.

## Why

- 바깥 진입면과 내부 운영 화면이 같은 제품으로 더 자연스럽게 읽힌다.
- 로그인 이후 화면의 브랜드 존재감이 너무 약하다는 이전 판단을 보완했다.
- lockup을 보조 위치에 두어 정보 우선순위는 유지했다.

## PM assessment

- 이번 배치는 저위험, 고효율이다.
- 브랜딩과 운영성의 균형이 괜찮다.
- 다음 단계는 더 많은 로고 노출이 아니라, 현재 수준을 유지한 채 실제 운영 흐름 품질을 더 올리는 쪽이 맞다.
