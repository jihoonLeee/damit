# Confirm Brand Integration Plan

## 목표
- 고객 확인 화면을 `다밋`의 운영 문서형 브랜드 톤으로 맞춘다.
- 고객 입장에서는 `무엇을 확인하는지`, `어디까지 읽으면 되는지`, `마지막 액션이 무엇인지`가 더 쉽게 읽히게 만든다.

## 적용 원칙
- 내부 운영 화면처럼 복잡하게 만들지 않는다.
- `문서 확인서`와 `확인 완료` 액션 구조를 우선한다.
- 기존 기능 흐름은 유지한다.
  - 공개 링크 접근
  - 사진/범위/금액 확인
  - 메모 입력
  - 확인 완료 기록

## 이번 배치 범위
- `public/confirm.html`
- `public/styles.css`

## 기대 효과
- 고객 확인 페이지가 제품 전체 브랜드와 더 잘 맞는다.
- 카드 간격과 문서 위계가 더 분명해진다.
- CTA가 더 또렷하게 읽힌다.
