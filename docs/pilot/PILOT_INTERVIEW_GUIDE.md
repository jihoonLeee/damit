# PILOT_INTERVIEW_GUIDE

## 인터뷰 원칙

- 정답 유도 금지
- 사용자의 실제 표현을 그대로 적기
- `좋으셨어요?`보다 `어땠어요?`를 사용
- 막힌 이유를 기능 설명으로 덮지 않기

## 시작 질문

1. 평소 추가금이 생기면 고객에게 어떻게 설명하세요?
2. 추가금 설명할 때 제일 귀찮거나 스트레스 받는 순간은 언제인가요?
3. 기록을 남겨야겠다고 느끼는 순간은 보통 언제인가요?

## 사용 직후 질문

1. 첫 화면에서 뭘 먼저 해야 하는지 바로 이해됐나요?
2. 사진 기록부터 합의 기록까지 흐름이 자연스러웠나요?
3. 어디에서 가장 멈칫했나요?
4. 버튼이나 문구 중 헷갈렸던 게 있었나요?
5. 평소 하던 방식보다 빨라진 느낌이 있었나요?

## 설명 초안 관련 질문

1. 초안 문구가 실제로 고객에게 보낼 만했나요?
2. 너무 딱딱하거나 이상한 표현이 있었나요?
3. 본인 말투에 맞게 조금만 바꾸면 될 수준이었나요, 아니면 거의 다시 써야 했나요?
4. 초안이 있으면 설명이 쉬워진다고 느꼈나요?

## 복사/메신저 관련 질문

1. 복사 버튼을 바로 이해했나요?
2. 복사 후 카카오톡/문자로 넘기는 과정이 자연스러웠나요?
3. 이 기능이 실제 현장에서 자주 쓸 것 같나요?

## 합의 기록 관련 질문

1. 합의 기록을 남기는 게 번거로웠나요, 아니면 남길 만했나요?
2. 어떤 항목은 꼭 필요했고, 어떤 항목은 과했다고 느꼈나요?
3. 이 기록이 실제로 분쟁 방지나 정리용으로 도움이 될 것 같나요?

## 가격/가치 질문

1. 이 도구가 계속 쓸 만큼 가치 있다고 느껴졌나요?
2. 돈을 낸다면 어떤 기준으로 내실 것 같나요?
3. 월 결제보다 건당/팀당/월정액 중 뭐가 더 자연스럽나요?

## 마무리 질문

1. 내일부터 실제로 다시 써볼 생각이 있나요?
2. 계속 쓰게 만들려면 제일 먼저 고쳐야 할 한 가지는 뭔가요?
3. 이걸 다른 청소 사장님에게 추천할 수 있을 것 같나요?
