# RECRUITMENT_MESSAGE_TEMPLATES

## 목적

- 첫 파일럿 대상 사장님 1~2명을 부담 없이 섭외한다.
- 너무 길거나 제품 냄새가 강한 문구보다, `실제 업무 도와주는 테스트`처럼 느껴지게 한다.

## 1. 지인 소개형 짧은 메시지

안녕하세요. 입주청소 현장에서 `추가금 설명`이 생길 때 사진, 설명 문구, 합의 기록을 조금 더 빨리 정리해주는 간단한 도구를 테스트 중인데요.

실제 사장님이 20~30분 정도만 써보시고 불편한 점을 알려주시면 제품 개선에 큰 도움이 될 것 같아서 연락드렸습니다.

광고나 계약 목적은 아니고, 실제로 현장에서 쓸 만한지 보는 첫 테스트입니다. 가능하시면 최근 사례나 실제 현장 기준으로 같이 한번 봐주실 수 있을까요?

## 2. 당근/카톡 DM용 조금 더 캐주얼한 버전

안녕하세요. 입주청소 사장님들이 현장에서 `추가 비용 설명`할 때 쓰는 간단한 도구를 만들고 있어서요.

사진 기록 -> 추가 사유 정리 -> 고객 설명 문구 -> 합의 기록까지 모바일에서 빠르게 해보는 테스트인데, 실제 청소하시는 분 피드백이 꼭 필요해서 연락드렸습니다.

20~30분 정도만 봐주셔도 되고, 불편한 점 세게 말씀해주셔도 괜찮습니다. 혹시 테스트 도와주실 수 있을까요?

## 3. 소개받은 분에게 보내는 신뢰형 버전

안녕하세요. OO님 소개로 연락드렸습니다.

입주청소 현장에서 추가금 설명이 필요할 때, 사장님이 사진 찍고 설명 문구 만들고 기록 남기는 과정이 번거롭다는 이야기를 듣고 작은 도구를 테스트 중입니다.

아직 초기 버전이라 정말 현업 관점 피드백이 중요해서, 20~30분 정도만 사용해보시고 어떤 점이 별로인지 말씀해주시면 개선에 반영하려고 합니다.

혹시 가능하시면 일정 한번 여쭤봐도 될까요?

## 4. 응답이 왔을 때 보내는 후속 답장

감사합니다. 테스트는 어렵지 않고, 휴대폰으로 제가 링크 드리면 최근 사례 하나 또는 실제 현장 흐름 기준으로 같이 20~30분 정도만 보면 됩니다.

제가 보고 싶은 건 `진짜 편한지`, `설명이 빨라지는지`, `기록이 남길 만한지`라서 불편한 점은 편하게 말씀 주시면 됩니다.

가능하신 날짜/시간대 2개 정도만 주시면 맞춰보겠습니다.

## 5. 거절 부담 줄이는 문장

혹시 지금 시기가 바쁘시면 괜찮습니다. 나중에 조금 여유 있을 때 짧게라도 의견 부탁드릴 수 있으면 정말 감사하겠습니다.

## 사용 팁

- 첫 연락은 길지 않게 보낸다.
- `AI`, `자동화`, `SaaS` 같은 단어보다 `현장 설명 빨라지는 도구`처럼 말한다.
- `좋은 말씀`보다 `불편한 점 세게 말해달라`고 해야 실제 피드백이 더 잘 나온다.
