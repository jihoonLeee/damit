## 첫 파일럿 권장 구성

- 사용자 수: `1~2명`

- 실제 현장 또는 최근 실제 사례 기반 시나리오: `3~5건`

- 사장님이 이 도구를 다시 열 이유가 생기는가

- 현장 기록부터 합의까지 흐름이 끊기지 않는가
