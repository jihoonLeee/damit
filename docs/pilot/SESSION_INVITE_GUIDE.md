# SESSION_INVITE_GUIDE

## 목적

- 파일럿 세션 일정을 잡고, 사용자에게 테스트 맥락을 짧고 명확하게 전달한다.
- 테스트 전부터 `법적 계약 도구`로 오해하지 않게 한다.

## 일정 확정 메시지 예시

안녕하세요. 테스트 도와주셔서 감사합니다.

이번 테스트는 `입주청소 현장에서 추가금 설명과 기록을 얼마나 빠르게 할 수 있는지` 보는 짧은 파일럿이고, 시간은 20~30분 정도 생각하고 있습니다.

제가 링크를 드리면 휴대폰으로 같이 보면서 진행하고,
- 사진 기록
- 작업 건 연결
- 변경 금액 저장
- 설명 초안 생성
- 합의 기록 저장

이 흐름을 최근 사례 기준으로 같이 확인할 예정입니다.

편하게 써보시고, 별로인 부분은 바로 말씀해주시면 됩니다.

## 세션 전 전달할 안내

- 가능하면 본인 휴대폰으로 접속
- 최근 실제 사례 1건 또는 설명이 어려웠던 사례 1건 준비
- 카카오톡/문자 붙여넣기까지 가능하면 같이 확인
- 테스트 중 불편한 점은 바로 말해도 괜찮음

## 세션 시작 멘트

오늘은 제품 시연보다 `실제로 쓸 만한지`를 보려고 합니다.
잘 되는 것보다, 어디에서 귀찮고 막히는지가 더 중요해서 불편한 부분은 바로 말씀 부탁드립니다.

이 도구는 계약이나 법적 효력을 대신하는 게 아니라, 현장에서 `설명 빠르게 하기`와 `기록 남기기`를 돕는 용도입니다.

## 세션 종료 멘트

오늘 말씀해주신 내용은 다음 수정 우선순위 정하는 데 바로 반영하겠습니다.
특히 `다시 쓸지`, `돈 낼 가치가 있는지`, `가장 별로였던 한 가지`가 제일 중요해서 마지막에 그것만 한 번 더 여쭤볼게요.

## 일정 운영 팁

- 첫 세션은 1명만 잡는다.
- 하루에 2세션 이상 몰아잡지 않는다.
- 세션 사이 최소 30분은 비워 바로 회고를 남긴다.
- 가능하면 오전 1회, 오후 1회로 나눠 기억 왜곡을 줄인다.
