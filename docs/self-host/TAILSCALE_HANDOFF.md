# TAILSCALE_HANDOFF

## What may require your action

Tailscale setup can usually be automated up to the login step.

You may need to do one of these:

- open the browser auth link shown by `tailscale up`
- approve the device in your Tailscale admin panel
- optionally provide an auth key later if you want unattended node enrollment

## What I can do

- install Tailscale on the Ubuntu server
- enable and start `tailscaled`
- run status and IP checks
- verify the app through the Tailscale path after login is complete
