# SKILL LIFECYCLE

## 상태
- draft
- active
- review-needed
- deprecated
- archived

## 상태 전이
1. draft
   - 초안 생성
   - 최소 구조 확인
2. active
   - 실제 프로젝트에서 사용 가능
3. review-needed
   - 중복, 모호성, 품질 문제 발견
4. deprecated
   - 대체 Skill 존재, 신규 사용 중단
5. archived
   - 보관 전환

## 리뷰 트리거
- 같은 역할 Skill 이 2개 이상 생김
- 산출물 품질이 일관되지 않음
- description 만 보고 호출 시점이 모호함
- 출력 파일 경로가 불명확함
- 핸드오프 실패가 반복됨
