# HOW TO USE SKILL STEWARD

## 신규 Skill 생성 요청 예시
skill-steward 를 사용해서 새로운 research-agent Skill 을 설계해줘.
기존 Skill 과 중복되는지 먼저 검토하고,
필요하면 새 Skill 대신 기존 Skill 개선안으로 제안해줘.

## 기존 Skill 개선 요청 예시
skill-steward 를 사용해서 feature-spec Skill 을 더 구체적으로 개선해줘.
아래 항목을 보강해줘.
- 호출해야 하는 경우 / 하면 안 되는 경우
- 출력 파일 경로
- 핸드오프
- 금지사항
- 다른 Skill 과의 경계

## 리뷰 요청 예시
skill-steward 를 사용해서 skills/ 아래 모든 SKILL.md 를 리뷰해줘.
중복, 모호성, 출력 누락, 핸드오프 누락을 점검하고
docs/skills/SKILL_REVIEW_QUEUE.md 와 output/skill-reviews/ 에 결과를 정리해줘.
