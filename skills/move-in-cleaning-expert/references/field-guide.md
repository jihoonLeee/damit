# Field Guide

## 목적
- 입주청소 현장 관점에서 제품, UX, 카피를 검토할 때 반복적으로 참고할 판단 기준을 정리한다.
- 이 문서는 `추가금`, `범위 변경`, `사진 증빙`, `고객 설명`, `분쟁 포인트`를 중심으로 본다.

## 1. 현장 기본 흐름
1. 고객이 온라인 또는 전화로 대략 조건을 전달한다.
2. 사장님은 평수, 구조, 옵션, 오염도 가정을 바탕으로 1차 견적을 준다.
3. 현장 도착 후 실제 상태를 확인한다.
4. 사전 가정과 다른 요소가 있으면 범위 변경 또는 추가 작업 판단이 생긴다.
5. 고객에게 이유를 설명하고 진행 여부를 합의한다.
6. 작업 후 이견이 생기면 현장 사진, 설명 내역, 합의 내용이 중요해진다.

## 2. 추가금이 자주 붙는 대표 상황
- 심한 오염과 방치 상태
- 니코틴, 기름때, 곰팡이, 찌든때
- 스티커, 보양지, 실리콘, 페인트 잔재 제거
- 폐기물, 짐, 생활 흔적 정리
- 베란다, 창고, 다용도실, 외부 샷시 등 추가 범위
- 평수/구조 차이
- 고객이 말하지 않았던 옵션 또는 특수 요청

## 3. 고객이 자주 하는 반응
- `원래 청소에 포함되는 거 아닌가요?`
- `전화할 때는 그런 말 없었는데요?`
- `사진만 봐서는 잘 모르겠어요.`
- `지금 꼭 해야 하나요?`
- `왜 이 정도 금액이 더 붙죠?`

## 4. 좋은 설명 구조
- 현재 상태: 지금 현장에서 확인된 문제가 무엇인지
- 추가 작업: 기본 청소 외에 어떤 작업이 더 필요한지
- 이유: 왜 기존 범위로는 처리하기 어려운지
- 금액 변화: 얼마가 어떻게 달라지는지
- 선택: 진행/보류/제외 중 무엇을 원하는지

## 5. 제품이 꼭 남겨야 할 최소 증빙
- 현장 사진
- 추가 사유 카테고리
- 짧은 현장 메모
- 원래 견적 금액
- 변경 후 금액
- 고객에게 보여준 설명 문구 또는 핵심 요약
- 고객 반응 또는 확인 상태
- 가능하면 확인 채널과 시각

## 6. 카테고리 설계 팁
- 너무 넓은 표현만 두지 않는다.
- `심한 오염` 하나로 끝내지 말고 실제 반발이 잦은 이유를 드러낸다.
- 사장님이 1초 안에 고를 수 있는 언어를 쓴다.
- 고객에게 설명할 때도 납득 가능한 단어를 고른다.

## 7. UX 검토 포인트
- 사진을 가장 먼저 남길 수 있는가
- 기본 정보 입력 때문에 현장 기록이 늦어지지 않는가
- 설명 초안이 고객용 문장으로 바로 쓸 만한가
- 상태값이 내부 진행률만 보여주는지, 실제 고객 반응을 담는지 구분되는가
- `보류`, `작업 제외`, `협의 실패` 같은 손실 사례가 보이는가

## 8. PM 검토 포인트
- 기본 청소 범위와 추가 범위 경계가 제품에 드러나는가
- 사용자가 돈을 내는 이유가 `예쁜 관리`가 아니라 `분쟁 감소/추가금 방어`로 연결되는가
- 파일럿 지표가 실제 현장 가치와 연결되는가

## 9. 절대 놓치지 말 것
- 사진만으로는 분쟁이 줄지 않는다.
- 상태값만으로는 합의가 증명되지 않는다.
- 추가금은 금액보다 설명 실패가 더 큰 문제다.
- 입주청소는 업종 특성상 `기본 포함 범위` 오해가 매우 잦다.
