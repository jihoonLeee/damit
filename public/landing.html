Workflow

작업 흐름은 네 단계로 이어집니다

1. 현장 문제 기록

사진과 사유를 빠르게 남겨 현장 상황이 놓치지 않도록 합니다.

2. 작업 건 연결

새 작업 건을 만들거나 기존 작업 건에 연결해 하나의 케이스로 관리합니다.

3. 변경 견적 정리

기본 금액과 변경 금액을 비교하면서 설명 근거를 정리합니다.

4. 설명과 확인 정리

설명 초안, 고객 확인 링크, 합의 기록, 타임라인을 한 화면에서 이어서 관리합니다.
